Web Application for Fiber Segmentation


This API is based on the YOLOv8 code from Ultralytics. We have used Flask API for web based application. .

Files supported are the common formats of images and videos that YOLOv8 also support: ('tif', 'jpg', 'jpeg', 'webp',... and 'avi', 'gif', 'mp4', ...)

The algorithm will return to you a labeled image with all the objects detected/segmented by trained model using custom data.

Instructions:

1) Use the "Choose file" button to select the file

2) Press "Submit" button to upload a file for prediction

3) Image, prediction, and its summary in an Excel file are shown in the result
